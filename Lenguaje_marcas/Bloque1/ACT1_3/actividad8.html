XML
Lenguaje de marcado extensible usado para almacenar y compartir datos entre sistemas de forma estructurada y personalizable.
HTML
Lenguaje base para crear páginas web, estructurando contenido como texto, imágenes y enlaces para que los navegadores lo presenten.
W3C
Organización que desarrolla estándares web (como HTML y CSS) para asegurar compatibilidad y accesibilidad en la red.
RFC
Documentos que definen normas y métodos técnicos para el funcionamiento de internet, publicados por la Internet Engineering Task Force (IETF)
